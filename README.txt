Técnicas de alocação atavés de algoritmo genético - ATI

. Segue abaixo a distribuição dos ficheiros


Docs: local reservado para documentos de apoio do projeto
Evolution: pasta reservada para etapas do algoritmo genético( indivíduo, população, reprodução,  mutação e seleção
models : pasta com as entidades básicas servidor e função
Parser: pasta com metodos para tratar a captação de dados csv
Main.py arquivo inicial que realiza o algoritmo completo
Por onde Começar

Para iniciar, é necessário que exista um arquivo .csv de funções a serem alocadas e outra com os servidores cadastrados

Especificações Técnicas

Ambiente desenvolvida em na linguagem python utilizando a biblioteca “Panda” como motor para importação e manipulação de arquivos CSV 

Testes

Pode-se realizar os testes de main. Para isso basta criar dois arquivos CSVs

Instalação

Utiliza-se a versão 3.6.4 do python que pode ser adquirido no site https://www.python.org/downloads/release/python-364/
Além disso , é necessário a instalação da biblioteca, seguindo o tutorial em 
https://pandas.pydata.org/


É preciso ter um plano de implantação do sistema para evitar imprevistos. Todos os passos para implantar um sistema deve ser informados a fim de evitar imprevistos.
